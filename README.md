
# Product Profitability & Margin Performance Dashboard

This project is an interactive Streamlit dashboard that analyzes product profitability & Margin Performance using the Nassau Candy Distributor dataset.

## Features
- KPI Metrics (Gross Margin, Profit per Unit,revenue share,profit share,margin volatility)
- Product Margin Performance Leaderboard
- Profit Contribution Analysis
- Division Revenue vs Profit Comparison
- Cost-sales scatter plots
- Margin Risk Detection
- Pareto Profit Concentration Analysis

## Tools Used
- Python
- Streamlit
- Pandas
- Plotly
- Matplotlib

## Dashboard
Interactive analytics dashboard built for product profitability & Margin Performance insights.
